# IS IT Command Hub
### Renewal by Andersen — Inside Sales Information Technology Department

A centralized internal operations platform built for the Inside Sales IT team. 
Replaces manual spreadsheet dashboards, fragmented planning tools, and 
disconnected workflows with a single, modern hub.

---

## What's inside

- **Standup Dashboard** — Interactive weekly task management with live 
  checkpoint rings, completion tracking, and photo upload support
- **Weekly & Monthly Planner** — Consolidated recurring and one-off task 
  planning, replacing Microsoft Planner, To Do, and Loop
- **Monday.com Integration** — Live ticket queue, kanban board, and 
  real-time operational metrics
- **Knowledge Base** — Centralized IT documentation, SOPs, setup guides, 
  and internal resources
- **AI Agent Assist** — Natural language search across documentation and 
  intelligent answers for IT specialists
- **User Management** — Role-based access control with Owner, Editor, and 
  Standard User permissions. MFA enforced via Microsoft Authenticator or Duo 2FA
- **Notifications** — Checkpoint reminders, overdue task alerts, and 
  Monday.com ticket notifications
